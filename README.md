# 👨🏻‍💻 Lista de comandos:

- Help: `./gradlew tasks`
- Check if code is correctly formatted: `./gradlew ktlintCheck`
- Format code: `./gradlew ktlintFormat`
- Run for development: `./gradlew bootRun`
- Run unit tests: `./gradlew test`
- Generating docs file: `./gradlew dokkaHtml`
